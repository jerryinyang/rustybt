ℹ️ About This Dashboard

This dashboard provides real-time visualization of RustyBT's benchmark performance. Data is sourced from docs/performance/benchmark-history.json.

How to Use

  • Run benchmarks: pytest tests/benchmarks/test_backtest_performance.py --benchmark-only
  • Regenerate dashboard: python scripts/benchmarks/generate_dashboard.py
  • View regression alerts: python scripts/benchmarks/detect_regressions.py
  • Query history: python scripts/benchmarks/query_history.py --summary

Benchmark Scenarios

The suite tests 15 scenarios across:

  • Frequencies: Daily, Hourly, Minute
  • Complexities: Simple (2 indicators), Medium (4 indicators), Complex (10+ indicators)
  • Portfolio Sizes: 10, 20, 50, 100, 500 assets
  • Optimizations: Python-only vs. Rust-optimized
